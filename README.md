# Sara Lahrach - Portfolio

A minimalist, modern portfolio website built with Next.js, React, and Tailwind CSS.

## Features

- Clean, dark-themed design inspired by modern portfolio aesthetics
- Responsive layout with glassmorphism effects
- Interactive bottom navigation
- Smooth animations and transitions

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations (if used)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
